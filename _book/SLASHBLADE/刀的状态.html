2. 刀的状态

2. 刀的状态

  • 断刀(Broken)：耐久为0且刀刃不完整

    • 造成影响：

      1.耐久不会低于0以下，也不会炸裂

      2.无法使用EX功能-额外幻影剑

      3.攻击力下降，攻击判定范围缩小

      4.无法使用SA，幻影剑

      5.EX功能-自我修复效果变为：消耗1级的玩家等级，回复最大耐久的10%并增加20点耀魂值

    • 解除状态的方法：

      断刀状态下回满耐久，进入完整状态。

    • 不可共存的状态：

      完整状态

  • 完整(PerFect)：耐久不为0且刀刃完整

    • 造成影响：

      1.解除断刀状态下的影响

      2.EX功能-自我修复效果变为：消耗10点的玩家等级(点不是级)，回复1点耐久并增加10点耀魂值

    • 解除状态的方法：

      进入断刀状态。

    • 不可共存的状态：

      断刀状态

  • 封(Sealed)

  • 注1：该状态分为三种情况，不同情况造成影响不同，解除方法也不同

    封状态1：刀没有附魔

    • 造成影响：

      无

    • 解除封状态1的方法：

      将刀附魔，进入附魔状态。

    封状态2：刀处于封状态无法解除

    封状态3：持有他人绑定的刀

    • 造成影响：

      1.无法解除断刀状态进入完整状态

      2.无法进入附魔状态

    • 解除封状态2的方法：

      该状态下不同的刀解封方法并不一样，

      例如：末影龙掉落的断刀阎魔刀需要通过合成的方式才能解除该封状态。

      (具体方法在单独介绍刀的页面内有标注。)

    • 解除封状态3的方法：

      归还刀给其主人。

    ​

    • 不可共存的状态：

      附魔状态

  • 注2：鞘为特殊情况，永远处于封状态

  • 附魔(Enchanted)：刀带有附魔

    • 造成影响：

      1.解锁SA

      2.刀损坏或断刀时额外掉落耀魂薄片

      3.附魔状态下可进一步进入妖状态

    • 解除状态的方法：

      移除所有附魔。

    • 不可共存的状态：

      封状态

  • 妖(Bewitched)：刀处于附魔状态下且拥有刀铭

    • 造成影响：

      1.解锁部分EX功能

      部分EX功能不仅需要妖状态也需要满足其他条件才能开启

      因此这里仅为告知能够解锁EX功能，不作详细

      2.显示SA

      3.解锁幻影剑

    • 解除状态的方法：

      1.解除附魔状态。

      2.移除刀铭。

    • 不可共存的状态：

      无

  • 噬魂者(SoulEeater)：刀的耀魂值超过1000

    • 造成影响：

      1.耀魂值(ProudSoul)显示变为紫色

      2.解锁EX功能-即时修复

    • 解除状态的方法：

      耀魂值低于1000。

    • 不可共存的状态：

      无

  • 凶狠利刃(FiercerEdge)：刀的杀敌数超过1000

    • 造成影响：

      1.杀敌数(KillCount)显示变为红色

      2.解锁超SA

      3.伤害提升

      4.伤害所需评级降低(B-SS/SSS/Limit变为B-A/S-SSS/Limit)

    • 解除状态的方法：

      杀敌数低于1000。

    • 不可共存的状态：

      无

  • 无鞘(NoScabbard)：没有刀鞘

    • 造成影响：

      1.攻击系统变为MC原版攻击系统

      2.无法使用EX功能-自我修复

    • 解除状态的方法：

      该状态下不同的刀解封方法并不一样，

      例如：末影龙掉落的断刀阎魔刀需要通过合成的方式才能解除。

      (具体方法在单独介绍刀的页面内有标注)

    • 不可共存的状态：

      无

  • 诅咒(Cursed)：作者尚未完善该状态

    • 造成影响：

      刀刃将无法拔出（刀刃和刀鞘连在一起）

    • 解除状态的方法：

      无。

    • 不可共存的状态：

      无

注1：本页仅用于介绍状态，并非所有刀都存在以上状态，请勿套用在所有刀上。(单独介绍某一把刀的页面有标注刀拥有哪些状态)
